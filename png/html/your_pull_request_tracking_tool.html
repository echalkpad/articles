About

Get your Pull Requests done

Hacksboard is a Pull Request tracking tool for teams using Github + Code Review + Office TV

Submit your pull request

Submit your pull request like you use to do, whenever you want. No more chat begging asking for review.

Your request appears at Hacksboard

Your pull request is displayed at your office TV using Hacksboard so everyone gets noticed. No asks for review, no lost pull requests.

You get reviewed

Your pull request is reviewed by someone else in your team so you can merge or rework on the issue as soon as possible.
