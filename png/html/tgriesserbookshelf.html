Bookshelf is a node.js ORM with support for postgreSQL, mySQL, and SQLite3. It is built atop the Knex Query Builder, and is strongly influenced by the Model and Collection foundations of Backbone.js. It features transaction support, one-to-one, one-to-many, and many-to-many relations.

For Docs, License, Tests, FAQ, and other information, see: http://bookshelfjs.org.

To suggest a feature, report a bug, or general discussion: http://github.com/tgriesser/bookshelf/issues/
